## Projetos em destaque

### Projeto Final TripleTen — Teste A/B, Funil de Conversão e SQL

Projeto de análise de dados com foco em validação de experimento A/B, análise de funil de conversão, teste Z para proporções, análise exploratória e consultas SQL.

**Tecnologias:** Python, pandas, NumPy, scipy, SQL, Jupyter Notebook, Looker Studio.

---

### Dashboard de Indicadores — Looker Studio

Projeto de Business Intelligence para visualização de KPIs, funil de conversão, distribuição de grupos, resultado de teste A/B e recomendações executivas.

**Tecnologias:** Looker Studio, Google Sheets, análise de indicadores e visualização de dados.

---

### LeveMente — App de Autocuidado Emocional

Aplicativo mobile em Flutter voltado ao autocuidado emocional, com check-in diário, trilhas de bem-estar, missões, gamificação e suporte em momentos de crise.

**Tecnologias:** Flutter, Dart, Riverpod, GoRouter, Hive, Firebase Messaging.

---

### API Cadastro de Cliente

API em Java com Spring Boot para cadastro e atualização de dados de clientes, demonstrando fundamentos de backend, organização de projeto e versionamento.

**Tecnologias:** Java 17, Spring Boot, Maven, Lombok, Git e GitHub.
